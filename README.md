# WorkoutDev

Web application to setup, record, and track workout sessions progress.

## Technologies
- Node.js
- JavaScript
- HTML
- CSS
- MongoDB

## Setup

### Prerequisites

Before you begin, ensure you have met the following requirements:

- [Node.js](https://nodejs.org/en/download/prebuilt-installer) (version 20.15.0)

### Running the Server

1. Clone the repository:
    ```sh
    git clone https://github.com/VitorBarbi/workout-dev.git
    ```

2. Navigate to the project directory, install the dependencies, and start the server:
    ```sh
    cd workout-dev
    npm install
    npm run dev
    ```

3. Open your web browser and go to:
    ```
    http://localhost:3000
    ```
